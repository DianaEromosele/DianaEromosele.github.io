www.CategorizedTweets.com

The app categorizes tweets, helping people to get up to speed with a politician's real-time thoughts on trending issues

Categorized Tweets is a Ruby on Rails application with a Bootstrap framework installed on the front-end. The feature uses the Twitter REST API, Sunlight Foundation Congress API, bulk data from GovTrack.us and Javascript's ds.js library to support data visualization

  • Wrote the proprietary algorithm that scans the Twitter timelines of politicians and sorts qualifying tweets into 8 key categories

  • Wrote the logic to make data requests to the Twitter REST API and the Sunlight Foundation Congress APIs; used ancillary Ruby gems to gain access to parsing methods

  • Developed the logic to parse bulk data provided by GovTrack.us

  • Used jQuery UI animation methods to boost UX and tweaked an d3.js/AJAX process to incorporate a spinning wheel

  • Wrote the function connecting users to the Congress API to enable a zip code search that yields one's representatives in Congress

  • Tweaked the models & controllers to incorporate the Friendly_Id gem that uses a semantic primary key to create human-friendly URLs

  • Built the associations, models, controllers, routes and views for an intuitive UI that accesses the categorized tweets from both a politician page & an issue page
